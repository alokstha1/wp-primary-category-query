# wp-primary-category-query
This plugin is used on the front-end that provides ability to query for posts (and custom post types) based on their primary categories passed in the shortcode parameter.
 To display in the front page use shortcodes <b>[wpcq]</b> for default post and category, <b>[wpcq post_type="post-type-name" taxonomy="taxonomy-name"]</b> into the pages.
